# 💈 Barbearia - Landing Page

## 📌 Sobre o projeto
Este projeto consiste em uma landing page para uma barbearia, desenvolvida com o objetivo de praticar a estruturação de páginas web utilizando HTML e CSS.
A proposta foi criar uma interface simples, organizada e visualmente agradável, simulando um site real de apresentação de serviços.

---

## 🚀 Tecnologias utilizadas
- HTML
- CSS

---

## 🎯 Objetivo
- Praticar a criação de páginas web
- Melhorar a organização de código
- Desenvolver noções de estrutura e layout
- Dar os primeiros passos no desenvolvimento front-end

---

## 💻 Funcionalidades
- Página inicial com apresentação da barbearia  
- Seção de serviços  
- Informações de contato  
- Estrutura de navegação básica
